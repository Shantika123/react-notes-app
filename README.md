# React Notes App

This is a **Notes App** built using React.js. It allows users to create, edit, delete, search, and pin notes efficiently.

The application uses **browser Local Storage** to persist data, ensuring notes remain saved even after refreshing or closing the browser. It also includes features like **dark mode** and **note color customization** for an improved user experience.

---

## Features

* Add new notes
* Delete notes
* Edit existing notes
* Search notes in real-time
* Pin important notes
* Dark mode toggle
* Notes color selection
* Data stored in Local Storage

---

## Tech Stack

* React.js
* JavaScript (ES6)
* HTML5
* CSS3 / Tailwind CSS
* LocalStorage

---

## How to Run the Project

```bash
# Clone the repository
https://github.com/Shantika123/react-notes-app.git

# Navigate into the project folder
cd your-project-folder

# Install dependencies
npm install

# Start the development server
npm start
```

The application will run at:

```
http://localhost:3000
```

## Deployment

This project is deployed on Vercel.

## 🌐 Live Demo

[Click here to view the app](https://react-notes-app-two-indol.vercel.app/)
### Deployment Steps

1. Go to [https://vercel.com](https://vercel.com)
2. Login with GitHub
3. Click **New Project**
4. Import your repository
5. Configure:

   * Build Command: `npm run build`
   * Output Directory: `build`
6. Click **Deploy**
